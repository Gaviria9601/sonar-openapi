Noncompliant Code Example (OpenAPI 2)

swagger: '2.0'
info:
  version: 1.0.0
  title: Swagger Petstore
paths:
  /customer/transactions:
    path
    ...

Compliant Solution (OpenAPI 2)

swagger: '2.0'
info:
  version: 1.0.0
  title: Swagger Petstore
paths:
  /customer/transactions:
    put
    ...
